<summary>Section 1</summary>
  
#### Создать предложение во множественном числе
 
* Plural Nouns (countable) — существительное во множественном числе
* A / An / The / Ø — артикль: 
    * a / an (не используются с мн. ч.), 
    * the (используется), 
    * Ø (отсутствие артикля — для общих понятий)

> [!TIP]
>
> Pronoun + verb + (adjective) + noun + (rest of the sentence)
> 
> (The / Ø) + (adjective) + noun + verb + (rest)

> [!EXAMPLE]
> * Cats sleep. - Кошки спят. (`Ø + plural noun + verb`)
> * We see children in the park. - Мы видим детей в парке. (`Pronoun + verb + plural noun + rest`)
> * The children are playing in the park. - Дети играют в парке. (`The + plural noun + verb + rest`)
> * I eat potatoes. - Я ем картошку. (`Pronoun + verb + plural noun`)
> * Small birds sing in the morning. - Маленькие птицы поют утром. (`Ø + adjective + plural noun + verb + rest`)
> * They sing in the morning. - Они поют утром. (`Pronoun + verb + rest`)
> * The red apples taste sweet. - Красные яблоки сладкие на вкус. (`The + adjective + plural noun + verb + rest`)
> * They taste sweet. - Они сладкие на вкус. (`Pronoun + verb + adjective`)
> * Flowers are beautiful. - Цветы красивые. (`Ø + plural noun + verb + to be + adjective`)
> * They are beautiful. - Они красивые. (`Pronoun + verb + to be + adjective`)
> * The books are on the table. - Книги на столе. (`The + plural noun + verb + to be + prepositional phrase`)
> * They are on the table. - Они на столе. (`Pronoun + verb + to be + prepositional phrase`)

<summary>Section 2</summary>
 
#### Создать предложение в единственном числе, используя:
* Uncountable Nouns - Неисчисляемые существительные
* Countable Nouns - Исчисляемые существительные

> [!INFO]
> Неисчисляемые существительные в английском языке:
> * не имеют формы множественного числа
> * не используются с артиклем a / an
> * используются в единственном числе (глагол в Present Simple имеет окончание -s)


> [!TIP]
>
> ***Pronoun + verb + (a / an / the / Ø) + (adjective) + noun + (rest)***
>
> ***(A / An / The / Ø) + (adjective) + noun + verb + (rest)***


 
> [!EXAMPLE]
> * I drink water. - Я пью воду. (`Pronoun + verb + Ø + uncountable noun`)
> * She likes music. - Она любит музыку. (`Pronoun + verb + Ø + uncountable noun`)
> * We eat an apple. - Мы едим яблоко. (`Pronoun + verb + an + noun`)
> * He drinks cold water. - Он пьет холодную воду. (`Pronoun + verb + adjective + uncountable noun`)
> * I like the new class. - Мне нравится новый класс. (`Pronoun + verb + the + adjective + noun`)
> * Water is cold. - Вода холодная. (`Ø + noun + verb`)
> * The water is cold. - Вода холодная. (`The + noun + verb`)
> * A cat sleeps. - Кошка спит. (`A + noun + verb`)
> * A big dog barks. - Большая собака лает. (`A + adjective + noun + verb`)
> * Good bread tastes great. - Хороший хлеб вкусный. (`Ø + adjective + noun + verb + adjective`)

function validateSection1(input) {
  if (!window.nlp) return;
  const doc = nlp(input);
  
  const pattern1 = '#Pronoun #Verb (#Adjective|) #Noun';
  const pattern2 = '(the|) (#Adjective|) #Noun #Verb';
  
  const check = doc.match(pattern1).found || doc.match(pattern2).found;
  
  return check ? [] : [{ err: true, msg: "The sentence is not grammatical" }];
}

function validateSection2(input) {
  if (!window.nlp) return;
  const doc = nlp(input);
  
  // Тип 1: Pronoun + verb + (a/an/the/Ø) + (adjective) + noun
  const pattern1 = '#Pronoun #Verb (a|an|the|) (#Adjective|) #Noun';
  
  // Тип 2: (A/An/The/Ø) + (adjective) + noun + verb
  const pattern2 = '(a|an|the|) (#Adjective|) #Noun #Verb';
  
  const check = doc.match(pattern1).found || doc.match(pattern2).found;
  
  return check ? [] : [{ err: true, msg: "The sentence is not grammatical" }];
}
