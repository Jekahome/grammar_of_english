#### Section 4

* "to be" — инфинитив настоящего времени (Present Infinitive)
* Указательные предложения (This is..., These are..., That is..., Those are...)

Указательные местоимения вместе с `to be` (Present Infinitive) - звучит очень формально. Это выражает - долженствование, возможность или договоренность. Переводится: «Должно быть», «Предстоит быть», «Нужно быть». Эту форму часто используют в: официальных документах, инструкциях, правилах, расписаниях, обсуждении планов. Вместо "This is to be here" в жизни чаще скажут "This must be here" или "This should be here".

> [!TIP]
> This/That/These/Those + is/are + to be + (adjective / prepositional phrase / noun / adverb)
> 
> This/That/These/Those + is/are + noun + to be + (adjective / prepositional phrase / adverb)


> [!EXAMPLE]
> 
> Noun: (предполагает контекст)
> * This is to be a doctor. — Это должно быть врачом. (Например, эта кукла в игре должна играть роль врача).
> * That is to be a teacher. — Тому предстоит быть учителем. (Обсуждаем распределение ролей).
> 
> Adjective: (предполагает контекст)
> * This is to be happy. — Это должно быть счастливым. (Например, конец фильма должен быть счастливым).
> 
> Prepositional phrase: (предполагает контекст)
> * This is to be at home. — Это должно быть дома. (Эта вещь должна находиться дома, а не на работе).
> 
> Adverb: (только с наречиями места и предполагает контекст)
> * This is to be here. — Это должно быть здесь. (Инструкция по расстановке мебели).
> * That is to be there. — То должно быть там.
> 
> **Примеры с существительным:**
> * This is a man to be happy — Это человек, который должен быть счастливым.
> * That is a woman to be strong — Вон та женщина должна быть сильной.
> * These are children to be kind — Это дети, которые должны быть добрыми.
> * Those are students to be first — Вон те студенты должны быть первыми.
> * This is a friend to be with me — Это друг, который должен быть со мной.
> * That is a doctor to be here — Вон тот врач должен быть здесь.
> * This is someone to be in love — Это кто-то, кто должен быть влюблен.

### Как это использовать в `match()`

Можго комбинировать теги со словами:

* `doc.match('#Pronoun #Verb')` — найдет "I walk", "They ate".
* `doc.match('will be #Adjective')` — найдет "will be happy", "will be late".
* `doc.match('#Determiner #Adjective #Noun')` — найдет "a big dog".
* `doc.match('!#Verb')` — найдет всё, что **не** является глаголом.
* `doc.match('going to #Infinitive')` — найдет "going to eat", "going to sleep".

### Как посмотреть теги в консоли?

```javascript
nlp("Your sentence here").debug()

const doc = nlp(text);
const hasVerb = doc.verbs().found;
```
Она выведет дерево предложения с расставленными тегами над каждым словом.
