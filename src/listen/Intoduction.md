# Listening Skills Training

Развитие навыка распознавать и понимать английскую речь с помощью адаптивных аудио книг и диалогов. 
Диалоги на определенные темы уровня английского, для поэтапного усвоения языка.  

1. Для понимания аудио, сперва его необходимо сделать понятным вводом (English Comprehensible Input). С текстом из аудио, нужно предварительно ознакомиться, прочитав его и **выписав новые слова**. После того как внутренний Мир из книги будет понят, можно ответить на вопросы для самопроверки.

2. Далее есть два варианта прослушивания:
    * слушать по предложениям пытаясь все понять и при этом писать английский текст вслед за диктором, и повторяя в голос;
    * свободно слушать привыкая к диктору, не пытаясь понять все что сказано
