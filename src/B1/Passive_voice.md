# Passive voice (all tenses, with modals)
**Passive Voice (Страдательный залог)** — это конструкция, где подлежащее не выполняет действие, а **испытывает действие на себе**. Давайте систематизируем это по всем временам и с модальными глаголами.

### Главное отличие: Active vs. Passive

*   **Active Voice (Действительный залог)**: Подлежащее само выполняет действие.
    *   **The chef** cooks the dinner. (**Шеф** готовит ужин.)
*   **Passive Voice (Страдательный залог)**: Подлежащее является объектом действия.
    *   **The dinner** is cooked by the chef. (**Ужин** готовится шефом.)

**Формула образования Passive Voice:**
**be + V3 (Past Participle)**
Глагол **`be`** ставится в нужном времени, а **`V3`** (третья форма глагола) остается неизменной.



### 1. Passive Voice во всех временах (Tenses)

Здесь показано, как меняется глагол **`be`** в зависимости от времени.

#### Группа Present (Настоящее время)

| Время | Формула | Пример (Active -> Passive) |
| :--- | :--- | :--- |
| **Present Simple** | am/is/are + V3 | **They build** these houses. -> **These houses are built** (by them). |
| **Present Continuous** | am/is/are being + V3 | **He is washing** the car. -> **The car is being washed** (by him). |
| **Present Perfect** | have/has been + V3 | **She has written** the letter. -> **The letter has been written** (by her). |

#### Группа Past (Прошедшее время)

| Время | Формула | Пример (Active -> Passive) |
| :--- | :--- | :--- |
| **Past Simple** | was/were + V3 | **He fixed** the computer. -> **The computer was fixed** (by him). |
| **Past Continuous** | was/were being + V3 | **They were painting** the wall. -> **The wall was being painted** (by them). |
| **Past Perfect** | had been + V3 | **She had finished** the work by 5. -> **The work had been finished** by 5 (by her). |

#### Группа Future (Будущее время)

| Время | Формула | Пример (Active -> Passive) |
| :--- | :--- | :--- |
| **Future Simple** | will be + V3 | **They will sign** the contract tomorrow. -> **The contract will be signed** tomorrow (by them). |
| **Future Perfect** | will have been + V3 | **He will have completed** the project by Monday. -> **The project will have been completed** by Monday (by him). |

#### Модальные глаголы (Modals) в Passive Voice

Это очень простая и важная конструкция.
**Формула: Modal Verb + be + V3**

| Модальный глагол | Пример (Active -> Passive) |
| :--- | :--- |
| **can** | **You can do** it. -> **It can be done** (by you). |
| **could** | **He could have fixed** it. -> **It could have been fixed** (by him). |
| **must** | **Students must obey** the rules. -> **The rules must be obeyed** (by students). |
| **should** | **They should clean** this room. -> **This room should be cleaned** (by them). |
| **have to** | **You have to send** the email. -> **The email has to be sent** (by you). |
| **may / might** | **The manager may sign** the documents. -> **The documents may be signed** (by the manager). |

> **Обратите внимание на Perfect Infinitive для прошедшего времени с модальными глаголами:**
> *   **should have been + V3**: The letter **should have been sent** yesterday. (Письмо следовало отправить вчера.)
> *   **could have been + V3**: The mistake **could have been avoided**. (Ошибки можно было избежать.)



### 2. Когда используется Passive Voice?

1.  **Когда исполнитель действия неизвестен, очевиден или неважен.**
    *   **My wallet was stolen.** (Мой кошелек украли.) — Кто украл, неизвестно.
    *   **This house was built in 1890.** (Этот дом был построен в 1890 году.) — Кто строил, неважно для смысла предложения.
    *   **He was arrested.** (Его арестовали.) — Очевидно, что полицией.

2.  **Когда важно само действие или объект действия, а не исполнитель.**
    *   **A new hospital will be opened next week.** (Новая больница будет открыта на следующей неделе.) — Важнее факт открытия, а не то, кто ее откроет.

3.  **В официальных notices, инструкциях, научных текстах.**
    *   **Passengers are requested to fasten their seatbelts.** (Пассажиров просят пристегнуть ремни.)
    *   **The mixture is heated to 100°C.** (Смесь нагревается до 100°C.)



### 3. Как указать исполнителя действия (Agent)

Если исполнителя нужно указать, используется предлог **`by`**.
*   *"Harry Potter" was written **by J.K. Rowling**.*
*   *The telephone was invented **by Alexander Bell**.*

> **Важно!** Не всегда исполнитель выражается предлогом `by`. Если действие совершается с помощью инструмента, используется предлог **`with`**.
> *   *The door was opened **with a key**.* (Дверь была открыта *с помощью ключа*.)
> *   *The cake was decorated **with fresh berries**.* (Торт был украшен *свежими ягодами*.)



### 4. Особые случаи и важные замечания

1.  **Непереходные глаголы (глаголы без прямого дополнения)** **не могут** иметь форму Passive Voice.
    *   *Нельзя сказать:* ~~The accident was happened.~~ (Глагол `happen` не имеет прямого дополнения).
    *   *Правильно:* *The accident happened.*

2.  **Глаголы с двумя дополнениями** (например, `give`, `send`, `show`, `tell`).
    Такие предложения можно сделать пассивными двумя способами:
    *   *Active:* He gave **her** (косвенное дополнение) **a book** (прямое дополнение).
    *   **Passive 1 (более частый):** **She** was given a book (by him).
    *   **Passive 2:** **A book** was given to her (by him).

3.  **Глаголы восприятия и состояния** (`see`, `hear`, `know`, `like`, `love`, etc.) в Passive Voice часто используются с инфинитивом `to` или причастием.
    *   *He was seen **to leave** the building.* (Его видели, как он выходил из здания.)
    *   *She was heard **singing** in the shower.* (Ее слышали, как она пела в душе.)

### Сводная таблица: Образование Passive Voice

| Время / Модальный глагол | Формула | Пример |
| :--- | :--- | :--- |
| **Present Simple** | am/is/are + V3 | The room is cleaned every day. |
| **Present Continuous** | am/is/are being + V3 | The room is being cleaned now. |
| **Present Perfect** | have/has been + V3 | The room has been cleaned recently. |
| **Past Simple** | was/were + V3 | The room was cleaned yesterday. |
| **Past Continuous** | was/were being + V3 | The room was being cleaned when I arrived. |
| **Past Perfect** | had been + V3 | The room had been cleaned before I arrived. |
| **Future Simple** | will be + V3 | The room will be cleaned tomorrow. |
| **Future Perfect** | will have been + V3 | The room will have been cleaned by 5 pm. |
| **Модальные глаголы** | can/should/must... + be + V3 | The room must be cleaned. |

**Ключевой вывод:** Страдательный залог смещает фокус с того, *кто делает*, на то, *что делается*. Умение использовать его правильно делает вашу речь более аутентичной и соответствующей стилю ситуации.
